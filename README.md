# UniDx
Unity-like DirectX framework for learning purposes.
※ This project is primarily documented in Japanese.

UniDxは、UnityライクなDirectXフレームワークです。

## 用途
- DirectX 入門（特にUnity経験者）
- ゲームフレームワークの構造理解に
- フレームワーク作成の参考・たたき台として

## 動作環境
- Windows 10/11
- Visual Studio 2022
- DirectX 11

## 依存ライブラリ
- DirectXTK
- DirectXTex
- tinygltf
